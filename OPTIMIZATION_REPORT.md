## 🚀 如何使用

### 开发环境
```bash
npm run dev          # 启动开发服务器
npm run dev:debug    # 调试模式启动
npm run type-check   # TypeScript 类型检查
npm run lint         # 代码质量检查
```

### 生产环境
```bash
npm run build        # 构建生产版本
npm run preview      # 预览生产构建
npm run analyze      # 分析 Bundle 大小
```

### 测试和质量
```bash
npm run health-check # 完整的健康检查
npm run lint:fix     # 自动修复代码问题
```
